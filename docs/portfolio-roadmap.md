# Roadmap do Portfolio

- [ ] Auditar e remover dependencias de autenticacao publica e integracoes herdadas que nao fazem sentido para um portfolio
- [ ] Simplificar `App.jsx` para uma composicao minima de providers e roteamento
- [ ] Extrair o conteudo do portfolio para arquivos locais tipados em `src/content`
- [ ] Reorganizar a home em secoes reutilizaveis e componentes puros
- [ ] Definir estrategia visual final para hero, projetos, stack, processo e contato
- [ ] Configurar cliente Supabase dedicado para o projeto
- [ ] Modelar tabela `contact_messages` com validacao minima e politica segura
- [ ] Implementar formulario de contato com validacao, persistencia e feedback de envio
- [ ] Adicionar fallback local para projetos caso o Supabase esteja vazio ou indisponivel
- [ ] Aplicar otimizacoes de performance: lazy loading, compressao de imagens, revisao de fontes e motion
- [ ] Configurar SEO basico e metadados sociais
- [ ] Executar lint, build e revisao final da estrutura
